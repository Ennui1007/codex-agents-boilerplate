# Existing Project Prompt

Use this prompt after adding this boilerplate to a project that already exists.

```md
Do not implement or modify files yet.

Use Plan Mode.

Inspect this repository and propose updates to the `Repository-Specific Overrides` section of `AGENTS.md`.

Please identify:

1. Language
2. Framework
3. Package manager
4. Runtime
5. Database
6. Auth
7. Payments
8. Hosting or deployment target
9. Test framework
10. Lint command
11. Typecheck command
12. Build command
13. Dev command
14. Main directories
15. Existing architecture patterns
16. Existing test coverage patterns
17. Security-sensitive areas
18. Areas that appear historical, inconsistent, or risky
19. Project-specific rules that should be added to AGENTS.md

Constraints:
- Do not change code.
- Do not add dependencies.
- Do not run destructive commands.
- Do not infer production details from secrets.
- If information is unknown, mark it as unknown.
```
